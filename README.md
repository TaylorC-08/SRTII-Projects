**Summary:** \
Science, Research, & Technology (SRT) is a course sequence part of the Science & Mathematics Academy. This repository compiles many of the smaller projects in 3D modeling as well as the coding projects from this class.

**What is inside?** \
Arduino-Projects contains Arduino sketches from the Arduino unit of SRTII
CSharp-Projects conatins C# statistics assignments in Visual Studio
CAD-Projects contains developed models in Autodesk Fusion, including parametric design
PCB-Projects contains later Printed Circut Board projects in Autodesk Eagle

This Repository overall covers developed code and coursework. Larger scale projects from SRTII are in a standalone repository.
